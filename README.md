# API REST – Gestión de Libros (ASP.NET Core)

API desarrollada con ASP.NET Core para la gestión de libros mediante operaciones CRUD.

## Tecnologías
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server / LocalDB
- Swagger
- C#

## Funcionalidades

- Obtener todos los libros
- Obtener libro por id
- Crear nuevos libros
- Actualizar libros
- Eliminar libros

## Endpoints

GET /api/libros  
GET /api/libros/{id}  
POST /api/libros  
PUT /api/libros/{id}  
DELETE /api/libros/{id}

## Características

- Programación asíncrona (async/await)
- Persistencia de datos con Entity Framework Core
- Manejo de errores y códigos HTTP
- Documentación automática con Swagger

## Objetivo

Proyecto desarrollado para practicar el desarrollo de APIs REST con .NET y el acceso a datos mediante Entity Framework.
